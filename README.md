# Student Success Intelligence Platform

## Overview

This project develops an AI-powered educational analytics platform designed to predict student success, identify academic risks, segment student populations, and generate actionable insights for educational institutions.

The platform combines predictive modeling, clustering, anomaly detection, and educational analytics to support data-driven decision-making.

## Problem Statement

Educational institutions often struggle to identify students requiring academic support before performance declines significantly.

This project aims to provide predictive insights that help educators improve retention, graduation rates, and academic outcomes.

## Dataset

A synthetic educational dataset was designed to simulate realistic student academic, demographic, and behavioral information.

### Features Included
